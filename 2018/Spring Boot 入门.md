### Spring Boot 入门

#### Spring Boot 精要

- 自动配置
- 起步依赖：利用了传递依赖解析
- 命令行界面：Spring Boot的可选特性,借此你只需写代码就能完成完整的应用程序,
  无需传统项目构建。
- Actuator

#### Spring Boot CLI 安装

1. 去网站下载 http://repo.spring.io/release/org/springframework/boot/spring-boot-cli
2. 把这个bin目录添加到系统路径里
3. `spring --version` 检查是否安装成功
4. `spring shell` 命令补全提示
